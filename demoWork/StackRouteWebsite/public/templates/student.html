Speecy Installation

We will be working on a Linux environment therefore it is a best to work on a Virual Environment, so that we do not have compatibility issues and provide a clean and fast development environment

You need to check whether Virtualization is enabled on your processor, to check the same, you can use the Speccy. Speecy provides the detailed system information. If Virtualization is not enabled, you would need to reboot and enable it through your System BIOS.

  • Download - Speecy
